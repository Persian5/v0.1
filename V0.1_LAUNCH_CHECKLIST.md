# V0.1 Launch Checklist

**Target:** March 2026
**Model:** Module 1 free, $4.99/month for Module 2+
**Stack:** Next.js 14 + Supabase + Stripe + Vercel

---

## Confirmed Working (no action needed)

These systems have been verified through code review and audit. Do not re-investigate unless a bug surfaces.

- Auth: Supabase email/password, verification, session persistence, protected routes, AuthGuard, LessonRouteGuard
- Onboarding: welcome sequence, goal setting, level assessment, display name with uniqueness
- Lesson system: LessonRunner, all game types (Flashcard, Quiz, Input, Matching, Audio, Grammar, Story, FinalChallenge), flexible step structure
- XP system: idempotent awards (stepUid v2), optimistic updates, Supabase persistence, cache versioning, reconciliation
- Streak system: daily streak tracking via Supabase
- Progress: lesson completion, module completion, sequential unlocking, "continue where you left off"
- Review mode: 4 games (Memory, Audio Definitions, Matching Marathon, Word Rush), filter modal, daily XP cap (1000)
- Vocabulary tracking: mastery levels, SRS scheduling, remediation, hard words identification
- Dashboard: level, XP, streak, daily goal, leaderboard widget, vocabulary stats, resume learning
- Leaderboard: API with rate limiting (60/min), 2-min cache, XSS protection, podium UI, pagination
- Paywall: ModuleAccessService, PremiumLockModal, check-module-access API, BlurredPreviewContainer
- Stripe: sandbox checkout, webhook handling, subscription status checking, billing pages
- Database: schema, migrations, RLS policies, RPCs (award_xp_unified, update_streak)
- Security: CSP headers, X-Frame-Options, HSTS, input validation, rate limiting on sensitive endpoints
- Mobile: responsive layouts with Tailwind breakpoints, touch-friendly interactions
- Content: Module 1 (4 lessons), Module 2 (7 lessons), Module 3 content exists in curriculum.ts
