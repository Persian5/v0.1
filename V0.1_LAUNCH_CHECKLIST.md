# V0.1 Launch Checklist

**Target:** March 2026  
**Model:** Module 1 free. Full Access: $0.99 first month (beta), then $9.99/month for Modules 2–11.  
**Stack:** Next.js 14 + Supabase + Stripe + Vercel  
**Last updated:** February 26, 2026

---

## Confirmed Working

- Auth is fail-closed: unauthenticated users do not get premium content.
- Module premium flags are correct: Modules 2-11 have `requiresPremium: true`; Module 1 is free.
- Module and lesson pages currently fail closed on premium-access check errors.
- Stripe webhook signature verification exists.
- Review hub and review games now use the same vocabulary source.
- Lesson progress, XP, streak, dashboard, leaderboard, onboarding, and review systems all exist and are wired.
